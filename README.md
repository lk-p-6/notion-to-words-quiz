# Notion Lesson Coach

Мини веб-приложение, которое превращает заметки урока из Notion в персональную среду для обучения.

## Возможности

- Вставь ссылку на публичную страницу в Notion и приложение вытащит содержимое разделов:
  - **"תיקונים – Исправления"**
  - **"מילים חדשות – Новые слова"**
  - **"משפטים – Предложения"**
- Apple / ChatGPT стилистика интерфейса с прогресс-баром по разделам.
- Flashcards для новых слов с поддержкой повторения, статистики, клавиатурных шорткатов и сброса прогресса.
- Чек-листы для исправлений и предложений, чтобы отмечать изученные элементы.

## Запуск локально

1. Установи Node.js версии 18 или новее.
2. Создай интеграцию Notion и выдай ей доступ к нужной странице. Сохрани секрет в переменной окружения `NOTION_TOKEN`.
3. Запусти сервер:

   ```bash
   NOTION_TOKEN=secret_token_value npm run dev
   ```

4. Открой `http://localhost:3000` в браузере, вставь ссылку на страницу урока и нажми **Start Learning**.

> Приложение использует только стандартные модули Node.js и нативный `fetch`, поэтому дополнительных зависимостей устанавливать не нужно.

## Структура проекта

- `index.html` – точка входа клиентского приложения.
- `src/styles.css` – стили с Apple / ChatGPT эстетикой.
- `src/main.js` – клиентская логика интерфейса и состояний обучения.
- `server.js` – минимальный Node.js сервер, который отдаёт статику и ходит в Notion API.

## Полезные шорткаты в Flashcards

- `Пробел` – перевернуть карточку.
- `←` – отправить слово в «Повторение».
- `→` – отметить слово как «Выучено».
- `Backspace` – отменить предыдущий шаг.

Наслаждайся изучением языков!
